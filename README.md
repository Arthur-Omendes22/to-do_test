# 📝 Gerenciador de Tarefas (To-Do List)

Um sistema simples e funcional de lista de tarefas desenvolvido em **Python**. O programa permite que você organize seu dia a dia diretamente pelo terminal, com salvamento automático de dados.

---

## 🚀 Funcionalidades

O programa oferece uma interface de menu interativa com as seguintes opções:

* **Adicionar tarefa:** Registra novas atividades na lista.
* **Listar tarefas:** Exibe todas as tarefas com seus respectivos status (✔ para concluída, ✘ para pendente).
* **Concluir tarefa:** Permite marcar uma tarefa específica como finalizada.
* **Remover tarefa:** Exclui uma tarefa da lista permanentemente.
* **Persistência de dados:** Todas as alterações são salvas automaticamente em um arquivo `tarefas.json`.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [Python](https://www.python.org/)
* **Armazenamento:** JSON (JavaScript Object Notation) para persistência de dados local.

## 📋 Como Executar o Projeto

1.  **Pré-requisitos:**
    Certifique-se de ter o Python instalado em sua máquina.

2.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/Arthur-Omendes22/to-do_test.git](https://github.com/Arthur-Omendes22/to-do_test.git)
    ```

3.  **Acesse o diretório:**
    ```bash
    cd to-do_test
    ```

4.  **Inicie o programa:**
    ```bash
    python seu_arquivo.py
    ```
    *(Substitua `seu_arquivo.py` pelo nome real do arquivo .py que contém o código)*.

---

## 📂 Estrutura de Arquivos

* `main.py`: O código-fonte principal do programa.
* `tarefas.json`: Arquivo gerado automaticamente para armazenar as tarefas (não apague se quiser manter seu histórico!).
